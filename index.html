The Limit of RL Fine-Tuning on Reasoning LMs

Reinforcement Learning Enhances Sampling Efficiency Without Expanding Reasoning Capacity

  Paper   Code

Key Findings

  • Finding 1: RL fine-tuning improves sampling efficiency, not reasoning ability.
  • Finding 2: Base models already possess intrinsic reasoning capacity.
  • Finding 3: Scaling inference sampling achieves similar benefits without RL.

Q&A

Q1: Does RL improve reasoning?

No — it mainly reduces the number of samples required to reach the same reasoning accuracy.

Q2: Why call it “Limit of RLVR”?

Because RL fine-tuning reaches a plateau where reasoning depth does not increase further.
